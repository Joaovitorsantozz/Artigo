Sobre a Pesquisa

Durante o segundo bimestre no Colégio da Polícia Militar Cruz Azul (Unidade Sorocaba), foi dirigida um trabalho para os alunos do segundo ano do médio, que estavam cursando a disciplina de observatório de fênomenos biólogicos, no itinerário de exatas. O trabalho constitua em os alunos, 15 ao total, se dividirem em grupos de 3 integrantes cada um, com o intuito de realizar pesquisas científicas sobre um assunto não determinado. O tema foi livre para todos os grupos, deixando assim livre para a criatividade de cada um deles. O teor da pesquisa também foi deixado de acordo com a preferência do grupo, podendo ser uma pesquisa baseada em uma experiência, ou uma pesquisa baseada em contagem de dados, porém teria de ser realizadas por nós mesmos.

    Este não é um site oficial da instituição de ensino Cruz Azul, foi feito por mim, João Vitor dos Santos, no intuito de publicar nossos trabalhos de pesquisa científica, e facilitar consequentemente, sua vizualização por outros alunos e/ou professores. Foi utilizado neste site as seguintes tecnologias :
  • Node.js para rodar o servidor
  • HTML e CSS como linguagens de marcação
  • Javascript para realizar as animações
    • A visualização do código fonte pode ser vista por inteira apartir do meu repositório Github
